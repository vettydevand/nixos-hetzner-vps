# Contributing

We welcome contributions of all kinds, from bug reports and feature requests to documentation improvements and code contributions.

## Bug Reports

If you find a bug, please open an issue on [GitHub](https://github.com/your-username/nixos-hetzner-vps/issues). Please include as much detail as possible, including:

- A clear and concise description of the bug.
- The steps to reproduce the bug.
- The expected behavior.
- The actual behavior.
- Your system configuration (`flake.nix`).

## Feature Requests

If you have an idea for a new feature, please open an issue on [GitHub](https://github.com/your-username/nixos-hetzner-vps/issues). Please include a clear and concise description of the feature, as well as a justification for why it would be a good addition to the template.

## Documentation Improvements

If you find an error in the documentation or have an idea for how to improve it, please open a pull request on [GitHub](https://github.com/your-username/nixos-hetzner-vps/pulls).

## Code Contributions

We welcome code contributions of all kinds. Before you start working on a new feature, please open an issue on [GitHub](https://github.com/your-username/nixos-hetzner-vps/issues) to discuss it with the community.

When you are ready to submit your code, please open a pull request on [GitHub](https://github.com/your-username/nixos-hetzner-vps/pulls). Please make sure that your code is well-tested and that you have updated the documentation as necessary.
