# Community

This document provides an overview of the community and how to get involved.

## Communication Channels

- **Discord:** For real-time chat and support, join our [Discord server](https://discord.gg/your-invite).
- **GitHub Discussions:** For questions, feature requests, and general discussion, use [GitHub Discussions](https://github.com/your-username/nixos-hetzner-vps/discussions).
- **GitHub Issues:** For bug reports, use [GitHub Issues](https://github.com/your-username/nixos-hetzner-vps/issues).

## Contributing

We welcome contributions of all kinds, including bug reports, feature requests, documentation improvements, and code contributions.

For more information, see the [Contributing Guide](contributing.md).

## Code of Conduct

All members of the community are expected to follow the [Code of Conduct](code-of-conduct.md).
