# Introduction

This is an introduction to the NixOS Hetzner VPS template.

This template provides a solid foundation for building your own NixOS server on a Hetzner VPS. It includes a number of modules that you can use to configure your system, as well as a sample `flake.nix` file that you can use as a starting point.

## Features

- **Declarative Configuration:** Configure your entire system with a single file.
- **Reproducible Builds:** Ensure that your system is always in a known state.
- **Atomic Upgrades:** Upgrade your system without fear of breaking it.
- **ZFS Filesystem:** Protect your data with a modern, reliable filesystem.
- **LUKS Encryption:** Encrypt your root filesystem for added security.
- **Firewall:** Block all incoming traffic by default.
- **Monitoring:** Monitor your system with Prometheus and Grafana.
- **Nginx Web Server:** Serve your web applications with a powerful, lightweight web server.
- **ACME SSL Certificates:** Automatically generate SSL certificates from Let's Encrypt.
- **Automatic Upgrades:** Keep your system up-to-date with the latest security patches.
- **Backups:** Automatically back up your system to a remote server.
